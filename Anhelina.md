# Mini-project 
## Cryptocurrency Examining and Fourier Forecast

Авторами цього парного проєкту є Одарка Кобилянська та Ангеліна Олексюк. Його метою є колаборація з напарником, аналіз та виявлення закономірностей у даних, утриманих з API, та створення моделі, на основі якої можна формувати трейдингові рішення
Ми обрали криптовалютний ринок, а об'єктом аналізу стала цифрова валюта Solana (SOL), цінність якої в її інноваційних технологіях, швидкості блокченів та низьких комісіях. Read more [here](https://www.binance.com/uk-UA/square/post/394172178218)

#### Цей проєкт складається з трьох частин:
1. data manipulation
2. MA and P&L calculations
3. Implementation of Fourier transform

В першій частині використано REST API та документацію з Binance, надалі було продовжено з використанням бібліотек json, Pandas, NumPy, Matplotlib i requests


Опис процесу аналізу (які інструменти та бібліотеки використовувалися).
Короткі висновки щодо ефективності вашої торгової стратегії.


Цей проєкт містить файл binance_api.ipynb, який не несе смислового навантаження і був створений в цілях інтересу викостання бібліотеки [websocket](https://developers.binance.com/en/docs/products/spot/web-socket-streams) (або [гітхаб-версія](https://websocket-client.readthedocs.io/en/latest/))для запуску даних в реальному часі
